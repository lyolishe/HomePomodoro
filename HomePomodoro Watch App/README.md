# Home Pomodoro

Этот проект является pet-проектом с целью реализации Pomodoro-таймера для старых часов apple watch series 3.

## Ключевые особенности
- автономность, независимость от iphone
- возможность выставлять длительность помидорки через digital crown
- по завершению помидорки она записывается в календарь
